# Pendekatan-Numerik-dengan-Metode-Lagrange
Mini project metode numerik menggunakan Interpolasi Lagrange untuk melakukan pendekatan nilai fungsi pada titik tertentu, yaitu f(0,75).  Perhitungan dilakukan dengan membentuk polinom Lagrange dari beberapa titik data hingga diperoleh estimasi nilai fungsi secara numerik menggunakan Microsoft Excel.
